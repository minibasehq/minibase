## Usage

```sh
git clone --depth 1 https://github.com/explodinglabs/minibase myapp
cd myapp
```

Create an env file:

```sh
cp .example.env .env
```

Start Minibase:

```sh
docker compose up -d
```

Full documentation in the [wiki](https://github.com/explodinglabs/minibase/wiki).
